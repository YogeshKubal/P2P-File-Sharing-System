1) Extract the ZIP and cd to the directory . 
2) There are 3 files p2pserver.py, p2pclient.py and sqldata.py 
3) The server can be started on one machine by typing 'python p2pserver.py' in the terminal
4) Before starting the client in an another machine Client's own address and Server's address need to be updated in the file 
5) Make sure the firewalls are down by typing 'sudo ufw disable' to let the packets flows
6) Name of the table that is created in the database is called Listing
